# 기여 안내 (Contributing)

이 프로젝트는 개인이 혼자 유지보수하는 저장소입니다. **외부 Pull Request는 받지 않습니다**
— 이미 작업해서 올려주신 PR이라도 리뷰나 병합 없이 닫힐 수 있습니다. 시간 들여 작업하시기
전에 이 점 참고 부탁드립니다.

버그 리포트나 기능 제안은 [Issues](../../issues)로 남겨주시면 확인은 하겠지만, 반영이나
답변을 약속드리지는 않습니다.

---

This is a solo-maintained personal project. **Pull requests are not accepted** — even a
completed PR may be closed without review or merge, so please don't spend time on one
before checking here.

Bug reports and feature suggestions are welcome via [Issues](../../issues), though a
response or fix isn't guaranteed.
