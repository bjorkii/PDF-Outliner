# PDF Outliner

PDF 북마크(목차) 조작 기능과 빠르고 편리한 내용 조회를 목적으로 만든 PDF 뷰어입니다.

- **배포 환경**: Apple Silicon Mac, Intel Mac, Windows
- **개발 스택**: Rust · [egui](https://github.com/emilk/egui)/eframe (wgpu) · [pdfium-render](https://github.com/ajrcarey/pdfium-render) · [lopdf](https://github.com/J-F-Liu/lopdf)

## 기능

- **PDF 뷰어 최소 기능 제공**: 확대·축소·드래그 / 내용검색 / 문서 내 링크 이동 / 페이지 이동 히스토리 Undo·Redo
- **북마크 편집**: 추가 / 수정 / 이동 / 삭제 / 저장
- **CSV/Excel 가져오기·내보내기**: 북마크 트리를 내보내 별도 파일로 아카이빙 / 파일 상태에서 편하게 수정 후 가져와 PDF에 반영
- **기타 기능**
  - 앱 구동시 직전에 열었던 파일과 페이지로 열림
  - OS 환경 차이로 파일명 자소분리 발생시 자동 수정
  - 파일이 열린 상태에서 이름이 변경되면 같은 폴더 내에서 추적하여 앱에 연동, 찾지 못할 경우 '다른 이름으로 저장' 안내.
  - 앱이 비상 종료할 경우에 대비해 자동저장
- **제약 사항**: 가상머신(parallels, vmware, virtualbox, utm 등)에서는 실행되지 않습니다.

## 단축키

| Windows | OS X | 기능 |
|---|---|---|
| Ctrl+B | Cmd+B | 북마크 추가 |
| F2 | F2 | 북마크 이름 수정 |
| Delete | Delete | 북마크 삭제 |
| Ctrl+Z | Cmd+Z | 실행 취소 |
| Ctrl+Shift+Z | Cmd+Shift+Z | 다시 실행 |
| Ctrl+S | Cmd+S | 북마크를 PDF에 저장 |
| Ctrl+F | Cmd+F | 문서 내용 검색 |
| Ctrl+C | Cmd+C | 선택한 텍스트 복사 |
| Ctrl+[ | Cmd+[ | 이전 화면(뒤로가기) |
| Ctrl+] | Cmd+] | 다음 화면(앞으로가기) |
| Tab | Tab | 사이드바 ↔ 뷰어 포커스 전환 |
| C | C | 단일 페이지 ↔ 연속 스크롤 모드 전환 |  
| ↑ / ↓ | ↑ / ↓ | 북마크 순회(사이드바 포커스 시) |
| ← / → | ← / → | 페이지 이동(뷰어 포커스 시) *또는* 북마크 접기·펼치기(사이드바 포커스 시) |

## 다운로드

[Releases](https://github.com/bjorkii/PDF-Outliner/releases)에서 플랫폼에 맞는 zip을
받아 압축을 풀고 실행하세요.

개발자 등록 없이 배포하므로 최초 실행시 경고 문구가 출력됩니다. 다음과 같이 조치하면 이후 정상 실행됩니다.
- **macOS**: 최초 실행시 "확인되지 않은 개발자" 경고 &rarr; 이후 `시스템 설정` > `개인정보 보호 및 보안`으로 이동하여 하단에서 `그래도 열기` 선택 &rarr; 재실행
- **Windows**: 최초 실행시 "'알 수 없는 게시자' 또는 'PC가 보호되었습니다' 경고 &rarr; `추가 정보` 선택 &rarr; `실행` 선택

## 소스에서 빌드

```bash
git clone https://github.com/bjorkii/PDF-Outliner.git
cd PDF-Outliner
cargo build --workspace --release
```

pdfium 동적 라이브러리가 런타임에 필요합니다(릴리스 zip에는 이미 동봉됨). 직접 빌드해
실행할 때는 `PDFIUM_DYLIB_PATH` 환경변수로 경로를 지정하세요 —
[bblanchon/pdfium-binaries](https://github.com/bblanchon/pdfium-binaries)에서 플랫폼별
바이너리를 받을 수 있습니다.
